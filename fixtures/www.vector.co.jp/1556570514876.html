SSD_TURBO_BOOST

SSDを快適・安全に使うためのSSD高速化・総合支援ソフト。キャッシュで高速化を図ったり、ファイルチェックやメンテナンスを行ったりできる。最新版でIntel系以外のNVMe SSDにも対応した。

旧バージョンはIntel系NVMe SSDのみ対応となっていた為、利用を躊躇していたユーザには朗報となるだろう。

ただし、利用するにはWindows 10の64bit環境が必要。

Windows 7でも利用する事は可能だが、Microsoftが提供するNVMe用ドライバをインストールする必要がある。

さて、「SSD_TURBO_BOOST」は人気ソフト「プチフリバスター Duo drive」の後継にあたるソフトで、

  • Speed Up 高速化
  • Refresh リフレッシュ
  • Maintenance メンテナンス
  • Migration tool 引っ越しツール

の4つの側面で、SSDの高速化・快適化を支援し、問題を解決するSSD用統合ユーティリティだ。

多彩な機能がまとめられ、メイン画面から簡単に呼び出して利用する事ができる。

また、計測した利用状況からSSDの寿命を推定する機能等も搭載している。

「Speed Up 高速化」は、パソコンに搭載されたメモリの一部をキャッシュとして利用する事で、元々高速なSSDを更に高速にしようというもの。

スイッチをONにして「自動設定」ボタンを押すだけの簡単操作で、SSDの高速化が実現する。

自動設定では、

  • 安全性を優先した「セーフ設定」
  • 標準的な設定の「スタンダード設定」
  • キャッシュメモリの効果を最大限に引き出す「ヘビー設定」

の3種類から選択する事ができる。

メモリのキャッシュ割り当て量をユーザがカスタマイズする事も可能だ。
